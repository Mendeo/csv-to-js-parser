combineArrays(obj, newKey, arrayKeys, newArrayKeysopt) → {Array.<object>}

In the objects that the csvToObj function returns, the properties that are responsible for non-grouped columns contain arrays of the same length. Using the combineArrays function, you can convert these arrays to a single array of objects, which in some cases may be a more natural representation of data in JavaScript.
Parameters:
Name Type Attributes Description
obj Array.<object> input object (usually from csvToObj)
newKey string the name of the property in which the arrays will be combined
arrayKeys Array.<string> the names of the properties in the input object that are arrays
newArrayKeys Array.<string> <optional>
 if specified: the names of properties that override properties from arrayKeys, i.e. the output object will have newArrayKeys properties instead of arrayKeys properties

csvToObj(data, param1opt, param2opt) → {Array.<object>}

Converting csv data to an array of JavaScript objects. The delimeter and description parameters can be set in a different order: first description, then delimeter.
Parameters:
Name Type Attributes Default Description
data string csv table as a string
param1 string <optional>
 ',' input data column delimeter
param2 object <optional>
 {columnName1 : { type: 'string', 'group': 1 }, columnName2 : { type: 'string', 'group': 1 }, ...} description
Properties
Name Type Attributes Description
columnName object <repeatable>
 the name of this property must match the column header in the input data.
Properties
Name Type Description
type string describes the type of column values. Three options are supported: "number", "string" and "boolean".
group number the order of grouping. Have to be set only for the columns that need to be grouped. There must be at least one such column. The values of this property indicate the order in which the columns should be grouped. For example columns with "group: 1" are grouped first, then columns with "group: 2" are grouped inside the first grouping and so on. You can perceive the group property in the same way as a "Group By" construct in SQL.

objToCsv(obj, delimiteropt, rowDelimiteropt) → {string}

Reverse convert an array of objects from csvToObj to CSV text format.
Parameters:
Name Type Attributes Default Description
obj Array.<object> input array of objects (the format must match the one returned by csvToObj).
delimiter string <optional>
 ',' column delimiter in the output table. If not specified, then the default is comma ",".
rowDelimiter string <optional>
 '\n' rows separator. If not specified, then the default is "LF" (\n). For windows, it is reasonable to specify the "CRLF" delimiter (\r\n).
